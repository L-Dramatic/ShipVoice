# ShipVoice 三人录音任务说明

这份说明是发给组员直接执行的版本。

你们只需要完成下面三件事：

1. 按分配到的编号，把对应句子读出来。
2. 每条句子单独录成一个音频文件。
3. 把这些音频文件打包发给我。

其他事情都不用管：

- 不用管项目文件夹怎么放。
- 不用管表格怎么填。
- 不用管格式转换。
- 不用管后续怎么评测。

这些都由我来处理。

## 你们到底要交什么

每个人最后交给我一包音频文件就行。

推荐交付方式：

- 把自己负责的音频放进一个文件夹。
- 文件夹命名为自己的名字加编号范围。
- 再压缩成一个压缩包发给我。

例如：

```text
张三_A001-A017.zip
李四_A018-A034.zip
王五_A035-A050.zip
```

## 可以用什么录

最推荐：**直接用手机自带录音软件**。

可用方式：

- iPhone：语音备忘录
- Android：系统录音机
- 电脑：Audacity 或 Windows 自带录音机

手机最方便，也完全够用。

## 录音要求

1. 每条句子单独录，不要把多条录在同一个文件里。
2. 读清楚，正常语速，不要太快。
3. 只读表里的这句话，不要自己改词，不要加解释。
4. 如果读错了，这一条重新录。
5. 每条 3 到 8 秒左右即可。
6. 开头和结尾稍微留一点空白，不要一按录音马上开口，也不要还没说完就停。

## 文件怎么命名

最好直接命名成对应编号：

```text
A001.wav
A002.wav
A003.wav
```

如果手机默认导出的是 `.m4a`，也可以：

```text
A001.m4a
A002.m4a
```

重点不是一定要 `.wav`，重点是**编号要对**。

## 怎么发给我

必须用“文件”方式发，不要直接发微信语音消息。

可以这样交：

- QQ 文件
- 微信文件
- 网盘链接
- U 盘拷给我

只要最后给我的是原始录音文件就行。

## 噪声要求怎么理解

有三种要求：

- `quiet`：安静环境录，背景声尽量小
- `classroom`：正常室内环境录，可以有一点环境声
- `workshop_like`：接近车间环境，可以有风扇声、轻微白噪声，但人声必须清楚

简单理解：

- `quiet` 就找个安静地方录
- `classroom` 就正常宿舍/教室环境录
- `workshop_like` 可以在风扇旁边、空调声旁边、轻微背景噪声环境录

不要故意制造太大噪声。听不清就没意义。

## 完成标准

每个人只要做到下面几点，就算完成：

1. 自己负责的编号全部录完。
2. 每个编号对应一个单独音频文件。
3. 文件名和编号对应。
4. 打包发给我。
