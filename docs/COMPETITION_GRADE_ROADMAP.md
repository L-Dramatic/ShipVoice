# ShipVoice 比赛级路线

## 当前基础

项目已经具备前后端、真实 provider 接口、安全门控、RAG、后台审计、录音清单、评测脚本和 LoRA 实验。

## 比赛级增强

1. 真实端到端录音扩展到 100 条以上。
2. 替换低延迟中文 TTS，实现句级并行合成。
3. 扩展船厂规程知识库并加入来源可信度。
4. 增加提示注入、多轮追问和 ASR 错字对抗集。
5. 使用 PostgreSQL、RBAC、对象存储和任务队列升级后台。
6. 制作真实链路演示视频和技术海报。

## 不变原则

系统只展示真实 provider 能力；服务失败时必须暴露失败并保留审计记录。
